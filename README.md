The AutoFuelShield is an innovative safety system designed to prevent deadly post-accident fuel tank explosions. In many road crashes, victims suffer not only from the collision but also from fire caused by leaking petrol or diesel. Traditional fire extinguishers are reactive and depend on human presence, whereas AutoFuelShield provides a proactive, automatic solution.

The system works by detecting collision-induced pressure using a sensor, which sends signals to an Arduino Uno. This triggers the servo motors that inject the Emulsifying agent mixture into the fuel tank. A DC motor then agitates the mixture, converting the fuel into a non-flammable emulsion, thus neutralizing the risk of fire within seconds.
